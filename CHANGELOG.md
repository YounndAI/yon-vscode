# Changelog

All notable changes to the `yon` VS Code extension.

## [0.1.1] — Unreleased

- Internal version bump (marketplace publishing setup). No user-facing changes;

## [0.1.0] — 2026-03-02

- Initial release: full YON v2.0 syntax highlighting via TextMate grammar.
- 70+ embedded language blocks inside `@BEGIN`/`@END`.
- Dedicated `.yon` file icon.
- Snippets for common YON patterns.
- Comment toggling, bracket matching, and folding support.
